# DatepickerWrapper 
This package is a wrapper around Vue2-Datepicker.
 
## Documentation
Visit: [https://mengxiong10.github.io/](https://github.com/mengxiong10/vue2-datepicker#readme)

## Features & characteristics:
* Single date
* Date Range
* Formatting

## Install & basic usage

```bash
npm i @dcodegroup-au/vue-datepicker
```
